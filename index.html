높이 필터 ／ Threshold

100cm

설정한 높이보다 위의 부품을 숨깁니다. 바닥부터 층별로 보며 쌓을 때 사용하세요. (화면만 숨김 · 부품 수엔 영향 없음)

모드

도구

그룹 장착(연결)

기존과 떨어진 그룹의 ①바닥 조인트 → ②연결 조인트 → ③붙일 대상 순으로 클릭하면, 바닥을 기준으로 기울여 붙입니다.

사각형 방향

가로 = 파이프 길이, 세로 = 육면체 높이. 바닥형은 위로 쌓이고, 벽형은 바닥부터 섭니다.

줄자 방향

바닥을 드래그해 선을 긋고 길이를 입력하세요.

파이프 길이 ／ 육면체 한 변

길이를 고르고, 조인트를 클릭하면 방향 화살표가 나옵니다. 화살표를 클릭해 연결하세요.

육면체 높이

선택 파이프 길이 변경

육면체는 같은 방향 파이프를 함께 선택해 바꾸면 형태가 유지됩니다.

참고 박스 ／ 파이프 아님

가로·세로·높이(cm)를 입력하면 반투명 육면체를 추가합니다. 부품 수엔 포함되지 않고, 선택 모드에서 드래그해 옮깁니다.

조작

  • 드래그 = 회전 · 휠 = 확대/축소 · 우클릭 드래그 = 이동
  • 조인트 클릭 = 방향 화살표 표시 → 화살표 클릭해 배치
  • 파이프/조인트 클릭 = 선택 → Delete 또는 삭제 버튼
  • 빈 곳 클릭 = 선택 해제 · 화살표 닫기
  • 단축키: V = 선택 모드 · P = 파이프 모드

저장 / 불러오기

📐 최종 치수

치수는 조인트 마감 여유(축당 +5cm) 포함. 축 길이는 조인트 중심 기준입니다.

📦 사용 부품

특수 커넥터 ／ 수동 입력

지붕·안경은 경사·평행 커넥터라 자동 인식이 안 됩니다. 개수를 직접 세어 입력하세요.

🏠 지붕 0
👓 안경 0

조인트 판정 규칙

조인트는 한 지점에 꽂힌 파이프의 개수·방향으로 결정됩니다.
1개 → 1구 · 2개 일직선 → 1자 · 2개 꺾임 → 2구
3개 T형(일직선+수직) → T자 · 3개 직각코너 → 3구
4개 평면 십자 → +자 · 4개 입체 → 4구 · 5개 → 5구 · 6개 → 6구
